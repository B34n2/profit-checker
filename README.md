# profit-checker
Allows you to check any ethereum TXID (specifically NFTs) and return info such as:
- Collection stats
- breakeven amount
- profit

## Requirements
- VC++ Build Tools
- Python 3.x

## Instructions
1. pip install -r requirements.txt
2. pip install -U git+https://github.com/Rapptz/discord.py
3. Edit .env
4. Run main.py

## .env
```
EMBED_COLOR=
BOT_TOKEN=
BOT_PREFIX=
OS_API_KEY=
MORALIS_API_KEY=
GUILD_ID=
```
